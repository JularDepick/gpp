# gpp
为了方便使用已有的g++.exe编译器批量将C/C++语言源代码编译为.exe文件，我编写了这个程序。实际上，如果需要的话，它可以代理任何.exe文件的批量运行。

## Releases中包含：
- gpp源代码编译得到的.exe文件，您可以尝试拖动多个源代码文件到它的图标上释放，这将使用您已安装配置好的g++.exe编译器批量编译这些源代码。
